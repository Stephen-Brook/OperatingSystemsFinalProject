# CSCI-460 Final Project

## Run Instructions
By default the program generates 100 random processes and runs each algorithm on those processes.
```bash
python3 main.py
```

To specify a number of processes you can use a flag:
```bash
python3 main.py -n numProcesses
```

## Algorithm Implementation Instructions
1. Follow the structure defined in the Scheduler class in ```schedulers.py```
2. Use the existing algorithms as reference
